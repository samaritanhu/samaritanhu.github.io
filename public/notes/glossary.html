骨骼 / 骨性标志 · Bones

English中文专业名昵称/俗称一句话解释
Femur 股骨 大腿骨 大腿里那根最长最粗的骨头。
Greater trochanter 股骨大转子 大转子 / “胯” 大腿骨最上端外侧的骨性突起；臀中肌就止（附着）在这里。
Scapula 肩胛骨 蝴蝶骨 后背上方两片像翅膀的扁骨。
Clavicle 锁骨 锁骨 脖子下方横着的那根骨头（属附肢骨，不是中轴骨）。
Ischium / ischial tuberosity 坐骨 / 坐骨结节 坐骨 / 屁股尖 坐下时压在椅子上的骨头；腘绳肌的起点。
Vertebral column 脊柱 脊椎 一节一节叠起来的脊椎骨。
Carpals 腕骨 手腕骨 手腕处的一堆小骨头。
Axial skeleton 中轴骨 中轴骨 身体中线：头颅 + 脊柱 + 肋骨 + 胸骨。
Appendicular skeleton 附肢骨 四肢骨 四肢 + 肩带 + 骨盆带（挂在中轴骨上的部分）。
Hinge joint 屈戌关节 / 铰链关节 合页关节 只能一个方向屈伸的关节，比如肘、膝。
Ball-and-socket joint 球窝关节 球窝关节 能多方向活动的关节，比如肩、髋。

肌肉 · Muscles

English中文专业名昵称/俗称一句话解释
Gluteus maximus 臀大肌 屁股 / 蜜桃臀 最大的臀肌，负责伸髋（把大腿往后蹬）。
Gluteus medius 臀中肌 臀中 臀部外侧深层，稳定骨盆、防止走路塌胯；止于大转子。
Psoas / iliopsoas 髂腰肌 屈髋肌 深层屈髋肌，连接腰椎和大腿，久坐容易紧。
Rectus abdominis 腹直肌 腹肌 / 六块腹肌 / 王字肌 前腹正中的肌肉，负责躯干前屈。
Obliques (internal/external) 腹内斜肌 / 腹外斜肌 侧腹 / 人鱼线 腰两侧，负责躯干旋转和侧屈。
Transversus abdominis 腹横肌 深层核心 / “天然腰带” 最深层的腹肌，像束腰带一样稳定核心。
Erector spinae 竖脊肌 竖脊 / 背部大梁 沿脊柱两侧的长肌，负责伸展和稳定脊柱；核心的后侧部分。
Quadratus lumborum (QL) 腰方肌 腰方 下背深层，负责侧屈和稳定腰部。
Multifidus 多裂肌 多裂 脊柱深层的小稳定肌。
Rectus femoris 股直肌 股四头肌前束 唯一同时跨过髋和膝的股四头肌（屈髋 + 伸膝）。
Biceps femoris 股二头肌 大腿后侧（外侧头） 腘绳肌的外侧头，有长短两个头。
Hamstrings 腘绳肌 大腿后侧 / 股后肌群 大腿后侧一组肌肉，负责屈膝、伸髋。
Quadriceps 股四头肌 大腿前侧 / 股四 大腿前侧四块肌肉，负责伸膝。
Pectoralis major 胸大肌 胸肌 胸部主要肌肉，紧了会含胸圆肩。
Latissimus dorsi 背阔肌 大背 / 倒三角 背部最宽的肌肉，练宽背靠它。
Infraspinatus 冈下肌 肩袖（之一） 肩袖四肌之一，负责肩关节外旋。
Rotator cuff 肩袖 肩袖肌群 稳定肩关节的四块深层小肌。
Gastrocnemius 腓肠肌 小腿肚 / 钻石小腿 小腿后侧大肌，负责绷脚（跖屈）。
Deltoid 三角肌 肩 / 饱满肩 肩膀轮廓的肌肉，分前中后束。
Trapezius 斜方肌 斜方 上背和颈肩的大肌，耸肩用到它。

动作 / 方向 · Movements & Directions

English中文专业名昵称/俗称一句话解释
Flexion 屈 屈 关节角度变小（弯曲）。
Extension 伸 伸 关节角度变大（伸直）。
Abduction 外展 外展 肢体远离身体中线。记忆：aBduct = Bye 拜拜，走远。
Adduction 内收 内收 肢体靠近身体中线。记忆：ADD 加回到身体上。
Pronation 旋前 翻掌向下 前臂旋转让掌心朝下。
Supination 旋后 翻掌向上 掌心朝上，像端着一碗汤。
Dorsiflexion 背屈 勾脚 脚尖往上勾，朝向身体。
Plantarflexion 跖屈 绷脚 脚尖往下踩/绷。
Inversion 内翻 脚内翻 脚底转向内，两脚掌相对。
Eversion 外翻 脚外翻 脚底转向外。
Internal rotation 内旋 内旋 旋转时前表面转向中线。
External rotation 外旋 外旋 旋转时前表面转离中线。
Elevation 上提 耸肩（上提） 身体部位往上抬（如耸肩）。
Depression 下降 沉肩（下降） 身体部位往下降（如沉肩）。
Circumduction 环转 绕圈 画圆的复合动作。
Prone 俯卧 趴着 / 脸朝下 脸朝下趴着。
Supine 仰卧 躺平 / 脸朝上 脸朝上躺着。
Superior / Inferior 上方 / 下方 上 / 下 靠上 / 靠下。肩在髋的上方（superior）。
Anterior / Posterior 前侧 / 后侧 前 / 后 身体前面 / 后面。
Medial / Lateral 内侧 / 外侧 内 / 外 靠近中线 / 远离中线。

体态 / 状况 · Posture & Conditions

English中文专业名昵称/俗称一句话解释
Anterior pelvic tilt 骨盆前倾 骨盆前倾（翘臀塌腰） 骨盆向前转、腰椎过度前凸、屁股翘起。久坐人群常见。
Posterior pelvic tilt 骨盆后倾 骨盆后倾 骨盆向后转、腰变平、皮带扣朝肚脐翘。
Neutral spine 脊柱中立位 中立位 脊柱最省力的平衡位置（保持自然曲度）。
Kyphosis 脊柱后凸 驼背 / 圆背 / 含胸 胸椎过度前弯，肩向前，即圆肩驼背。
Lordosis / hyperlordosis 腰椎前凸 / 过度前凸 塌腰 腰椎曲度过大（常伴骨盆前倾）。
ASIS 髂前上棘 髂前上棘 骨盆前方摸得到的骨性突起，用来判断骨盆前后倾。
Stenosis 椎管狭窄 椎管狭窄 脊髓通道变窄；后伸会更窄（加重），前屈能缓解。
Herniated disc 椎间盘突出 突出 / 膨出 椎间盘往外突压到神经；忌反复前屈和旋转。
Impingement 撞击 / 夹挤 肩夹挤 关节间隙变窄，里面的结构被挤压。

生理 / 能量系统 · Physiology & Energy

English中文专业名昵称/俗称一句话解释
Agonist / Antagonist 主动肌 / 拮抗肌 发力肌 / 对抗肌 主动肌发力做动作，拮抗肌在对侧。主动肌紧→拮抗肌常弱。
Concentric 向心收缩 向心 肌肉缩短并发力（举起阶段）。
Eccentric 离心收缩 离心 肌肉被拉长同时控制（放下阶段）。
Isometric 等长收缩 等长 / 静力 长度不变、关节不动地发力（平板支撑、推墙）。
Phosphagen system 磷酸原系统 ATP-CP 系统 开头几秒的爆发供能（如起跳、冲刺起动）。
Glycolysis 糖酵解 无氧糖酵解 靠肌糖原（碳水）供能，中高强度。
Oxidative system 氧化系统 有氧系统 需要氧气，烧脂肪和碳水，长时间低强度。
Catabolic state 分解代谢状态 分解态 / 掉肌肉 身体在分解组织（练太多、没恢复），需要休息和补给。
Golgi tendon organ (GTO) 高尔基腱器 腱器 感受“张力”，张力过大时让肌肉放松（保护）。
Muscle spindle 肌梭 肌梭 感受“被拉伸”，触发牵张反射让肌肉收缩（防拉伤）。
Proprioception 本体感觉 本体感 身体对自身位置和动作的感知（闭眼也知道手在哪）。
Neuromuscular junction 神经肌肉接头 神经肌肉接头 神经和肌肉相接、传递信号的地方。
